Install Veil

Chrome and Microsoft Edge extension. Firefox supported with Gecko ID configuration.

Personal use

  1. Install from your browser store (or load unpacked for pilots — see below).
  2. Open Veil → choose Personal → set a passphrase.
  3. Highlight sensitive text in Gmail, Outlook, or any web form → Secure selection or use the keyboard shortcut.

Team members

  1. Install Veil from your org catalog or store link from your admin.
  2. Open the popup → Team → enter join code and work email.
  3. Save the team passphrase your admin shares (password manager recommended).
  4. Refresh Outlook or Gmail. Veil copilot appears on paste and while typing sensitive content.

Recipients without Veil

When you secure a message for someone outside your org, Veil can embed a link to the hosted unlock page. They enter the passphrase you shared out of band — decryption runs in their browser; nothing is uploaded.

For IT administrators

  • Deploy via Chrome / Edge managed policy with extension ID and JSON policy (team passphrase, copilot defaults).
  • Apply a policy pack in the admin portal — Finance, Healthcare, Engineering, or observational rollout.
  • Forward metadata-only security events to your SIEM webhook; export JSON or CSV from the admin dashboard.

Enterprise deployment details: extension/docs/ENTERPRISE.md in the Veil repository.

Developer / pilot install

  1. Run npm run package in the Veil repository.
  2. Chrome or Edge → Extensions → Developer mode → Load unpacked → extension/dist.
  3. Reload after each update.

Supported browsers

Chrome 128+, Microsoft Edge 128+, Firefox 128+ (Gecko extension ID required).

Each browser profile keeps its own Veil setup. Join your team separately in Edge and Chrome if you use both.
