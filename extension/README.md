# Veil by Goldspire

Browser extension for securing sensitive text in Outlook, Gmail, and web apps.

## Quick start

1. Load unpacked from `extension/dist` (after `npm run package`)
2. `Ctrl+Shift+S` or right-click → **Veil → Secure selection**
